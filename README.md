SmartGov Hub |demo:https://smartgovhub.web.app

SmartGov Hub is a centralized web platform developed to simplify access to government schemes and services. The platform helps users easily search, explore, and discover verified schemes through intelligent recommendations, notifications, and AI-assisted guidance.

────────────────────────────────────────

Project Objective

Government scheme information is often scattered across multiple websites with different interfaces and navigation systems. This creates confusion for users and makes it difficult to identify authentic and relevant schemes.

SmartGov Hub addresses this challenge by providing a single platform where users can securely access, search, and explore schemes in a simple and organized manner.

System Modules

Authentication Module
Handles secure user registration and login using Firebase Authentication.

Search Module
Allows users to search schemes dynamically using keywords and categories.

Scheme Management Module
Stores and manages government scheme data using Firebase Firestore.

Recommendation Module
Analyzes user activity and interests to suggest relevant schemes.

Notification Module
Displays popup notifications and important scheme updates.

AI-Assisted Chatbot Module
Provides intelligent guidance to help users discover suitable schemes based on queries, categories, and interests.

Admin Module
Allows administrators to add, update, and manage scheme-related information.

────────────────────────────────────────

System Workflow

1. Users create an account and log in securely.
2. Users search schemes using keywords or categories.
3. Scheme data is retrieved from Firebase Firestore.
4. User activity is analyzed for personalized recommendations.
5. Popup notifications display relevant updates periodically.
6. The AI-assisted chatbot helps users discover suitable schemes.
7. Administrators manage and maintain scheme information.
